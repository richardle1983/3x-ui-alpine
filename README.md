# 3x-ui-alpines项目说明: 
本项目从3x-ui官方版本移植到Alpine系统, 面板功能与官方版本完全一致, 面板使用说明请参考官方文档.
```
https://github.com/MHSanaei/3x-ui
```

# 一键安装
```
bash <(curl -Ls https://raw.githubusercontent.com/56idc/3x-ui-alpine/master/install_alpine.sh)
```
